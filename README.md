## 🚧 Work in Progress — But Already Thinking

What if your Minecraft server didn't just run...

What if it **watched. Learned. Spoke.**

**MCmadeEasy** is a smart Minecraft server management and local hosting client powered by **Groq AI** — built to turn your server into something that feels alive.

This isn't just a control panel.

It's an **intelligent overseer for your world.**

---

# 🎮 Currently Working Server Types

MCmadeEasy supports **any Minecraft server JAR**.

• **Paper** – High-performance Spigot fork  
• **Purpur** – Advanced customizable Paper fork  
• **Fabric** – Lightweight modding platform  
• **Forge** – Popular modding framework  
• **Vanilla** – Official Minecraft server  
• **NeoForge** – *(Integration in progress)*  

Just select your preferred server type when creating a profile.

# 🎯 Natural Language Command Execution

No more memorizing complex commands.

Just say:

* "Give me 5 diamond blocks."
* "Teleport me to nearest village."
* "Set time to night."
* "Give me 64 dirt and 4 obsidian."

The AI understands **plain English** and converts it into **proper Minecraft commands** — even executing **multiple commands at once (For Give type commands)**.

# 🖥️ Modern Dashboard

A clean, dark-themed dashboard inspired by modern hosting panels.

Features include:

* Multiple server profile management
* Server type selection (Paper, Fabric, Forge, Spigot, etc.)
* RAM allocation per profile
* Server settings editor with sliders
* AI personality & system prompt configuration
* Role-based permissions
* Live console with command support
* Embedded Dynmap viewer
  
> **Note**  
> The UI design is mostly inspired by **Aternos**, the free Minecraft server hosting platform.

# 🚀 Quick Start

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the server:

```bash
python -m uvicorn backend.main:app --port 8001
```

Open in browser:

```
http://localhost:8001
```

Your **intelligent Minecraft server manager** will start.
